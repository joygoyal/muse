# Local Music Files

Put your private MP3 files in this folder with these filenames:

- `golden-hour.mp3`
- `until-i-found-you.mp3`
- `mrignaini-karun.mp3`
- `her-jvke.mp3`

The loop timing and lyric/caption lines are controlled from `data/songs.json`.

`muse-preview.wav` is the bundled original fallback preview used when a private
MP3 has not been added yet or the browser blocks scripted playback.
